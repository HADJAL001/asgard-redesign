# Google Play — OSGARD

## Название приложения (30 символов)
OSGARD

## Краткое описание (80 символов)
Генератор мифологических артефактов по описанию или голосу, на базе ИИ

## Полное описание (4000 символов)

OSGARD — генератор персональных мифологических артефактов. Опиши словами
или голосом предмет силы — меч, амулет, реликвию — и получи уникальный
артефакт с редкостью, характеристиками и историей, сгенерированный ИИ.

Возможности:
• Создание артефактов по текстовому или голосовому описанию
• Система редкости и типов артефактов
• История всех созданных артефактов с фильтрами по типу, редкости, дате
• Внутренний баланс и стоимость генерации
• Push-уведомления о готовности артефакта
• Офлайн-доступ к профилю, кошельку и истории артефактов — данные
  остаются доступны без интернета, синхронизация происходит автоматически
  при восстановлении сети

OSGARD подойдёт тем, кто любит мифологию, фэнтези-миры и коллекционирование
уникальных предметов.

## Графика (готовится отдельно, не текстом)
- Иконка приложения 512×512 (адаптивная, PNG, без прозрачности для фона) —
  экспортируется из `mobile/assets/images/icon.png` (уже брендирован, см.
  `mobile/scripts/generate-brand-assets.js`), при экспорте под Play Console
  убрать прозрачность/привести к требуемому размеру.
- Featured graphic 1024×500
- Скриншоты телефона: минимум 2, до 8, соотношение 16:9 или 9:16
- Скриншоты планшета (опционально)

Скриншоты снимаются с реального билда (девелопмент-клиент или preview-APK
из `eas build --profile preview`) на устройстве/эмуляторе — автоматический
захват через `expo start --web` в этой среде заблокирован pre-existing
проблемой с `react-native-worklets` (см. `mobile/RELEASE.md`).
